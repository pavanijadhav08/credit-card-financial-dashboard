# Credit Card Financial Performance & Customer Analytics Dashboard (SQL + Power BI)
## Project Objective
To design and develop an interactive Credit Card Financial Dashboard that provides comprehensive insights into revenue performance, transaction trends, customer demographics, and card category analysis using SQL and Power BI.

The objective of this project is to:
Monitor total revenue, interest earned, and transaction activity.
Analyze quarterly performance and growth trends.
Identify high-value customer segments based on age, gender, income, education, and profession.
Evaluate card category performance and acquisition cost.
Examine spending behavior across different expenditure types.
Track geographic contribution by top-performing states.
Support data-driven decision-making for improving profitability and customer strategy.

## Questions (KPIs)
- What is the total revenue generated by the credit card business?
- How much total interest income is earned from customers?
- What is the total transaction amount and total transaction count?
- Which quarter shows the highest revenue and transaction performance?
- Which gender and age group contribute the most to overall revenue?
- Which credit card category generates the highest revenue?
- Which states are the top contributors to total revenue?
- Which customer job category contributes the most revenue?
- Which transaction method (Swipe, Chip, Online) is most frequently used and generates the highest revenue?

## Process 
- Collected customer and transaction datasets in CSV format.
- Imported data into SQL and created relational tables.
- Connected SQL database to Power BI.
- Cleaned and transformed data (handled nulls, created age groups, calculated revenue, extracted quarters).
- Created DAX measures for revenue, interest, transactions, and customer analysis.
- Built interactive dashboards to analyze financial performance, customer segments, card categories, and transaction trends.

## Project Insights
- Total revenue reached 55.4M with strong performance in Q3.
- Female customers contributed slightly higher revenue than males.
- Age group 40–50 is the most profitable segment.
- Blue card category dominates overall revenue contribution.
- Majority transactions are done through Swipe method.
- Revenue is concentrated in top states like TX, NY, and CA.


## Final Conclusion
The dashboard provides a clear 360° view of credit card financial and customer performance. It helps identify high-value customer segments, top-performing card categories, and key revenue-driving regions. The insights support data-driven decision-making to improve profitability, optimize acquisition strategies, and monitor business growth effectively.
