# Credit Card Financial Performance & Customer Analytics Dashboard (SQL + Power BI)
## Project Objective
To design and develop an interactive Credit Card Financial Dashboard that provides comprehensive insights into revenue performance, transaction trends, customer demographics, and card category analysis using SQL and Power BI.

The objective of this project is to:
Monitor total revenue, interest earned, and transaction activity.
Analyze quarterly performance and growth trends.
Identify high-value customer segments based on age, gender, income, education, and profession.
Evaluate card category performance and acquisition cost.
Examine spending behavior across different expenditure types.
Track geographic contribution by top-performing states.
Support data-driven decision-making for improving profitability and customer strategy.
